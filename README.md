## Environment setup
If webpack is not installed

```sh
  $ npm webpack -g
```
then:

```sh
  $ cd code-test
```

```sh
  $ npm install
```

## Development

Start webpack server to view:

```sh
  $ npm start
```

Open [http://localhost:3001](http://localhost:3001)

or:

Navigate to the the index.html file at src/index.html and just open it in chrome
