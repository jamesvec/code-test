## Environment setup
If webpack is not installed

```sh
  $ npm webpack -g
```
then:

```sh
  $ cd code-test
```

```sh
  $ npm install
```

## Development

Bundle assets and then start webpack server to view:

```sh
  $ npm start
```
then open a new terminal window and run:

```sh
  $ npm run bundle
```

Open [http://localhost:3001](http://localhost:3001)
